## 1. 서비스 개요

### 1.1 서비스명
**GiftPick** (기프트픽)

### 1.2 서비스 한 줄 정의
> "선물 주는 사람의 고민을 없애고, 선물 받는 사람의 만족을 극대화하는 선물 큐레이션 플랫폼"

### 1.3 서비스 배경 및 문제 인식

현대인들이 선물을 준비할 때 마주하는 공통적인 어려움:

| 문제 | 설명 |
|------|------|
| **취향 불일치** | 상대방의 취향을 정확히 알지 못해 원치 않는 선물을 구매하는 경우 발생 |
| **중복 선물** | 이미 가지고 있는 물건을 선물하거나, 여러 명이 같은 선물을 준비하는 상황 |
| **시간 낭비** | 선물 탐색, 비교, 결정에 과도한 시간을 투자 |
| **받는 사람의 부담** | 원하지 않는 선물을 받았을 때 감사 표현과 처리에 대한 심리적 부담 |

### 1.4 서비스 목표
- 선물 주는 사람(Giver)의 선물 선택 고민 **제로화**
- 선물 받는 사람(Recipient)의 선물 만족도 **극대화**
- 선물 이벤트 관리의 **디지털화 및 간편화**

## 2. 핵심 가치 제안 (Value Proposition)

### 2.1 Giver (선물 주는 사람)를 위한 가치

```
✅ 선물 고민 해소     → 수혜자가 직접 원하는 선물을 선택
✅ 편리한 이벤트 관리  → 이벤트 생성·옵션 등록·현황 확인을 한 곳에서
✅ 개인화된 선물 경험  → 수혜자 취향 존중으로 의미 있는 선물 전달
✅ 시간 절약          → 선물 선정·구매 과정의 시간 최소화
```

### 2.2 Recipient (선물 받는 사람)를 위한 가치

```
✅ 선택의 자유        → 정말 원하는 선물을 직접 선택
✅ 만족도 극대화      → 원치 않는 선물 수령 방지
✅ 간편한 선택 과정   → 직관적 UI로 쉽고 빠르게 선택
✅ 심리적 부담 감소   → 선물에 대한 불필요한 부담 최소화
```

## 3. 서비스 사용자 정의

### 3.1 사용자 역할 (User Role)

| 역할 | 명칭 | 설명 |
|------|------|------|
| **선물 주는 사람** | Giver (관리자) | 선물 이벤트를 생성하고 선물 옵션을 등록, 수혜자를 초대하며 선택 결과를 확인 후 실제 구매 진행 |
| **선물 받는 사람** | Recipient (수혜자) | 초대 링크를 통해 플랫폼에 접속, 제공된 선물 옵션 중 하나를 선택 |

### 3.2 타겟 사용자 (Target Users)

**주요 타겟:**
- 20대 후반 ~ 40대 직장인 (온라인 서비스 친숙, 선물 고민 빈번)
- 연인·가족·친구·동료 등 다양한 관계에서 선물을 주고받는 사람

**세부 세그먼트:**

| 세그먼트 | 특징 | 니즈 |
|----------|------|------|
| 개인 Giver | 친밀한 1:1 관계에서 선물 | 수혜자 만족도 높은 선물 선택 |
| 단체 Giver | 직장 동료·모임 단체 선물 | 의견 조율 및 선택 효율화 |
| 선물 초보 | 취향 파악이 어려운 관계 | 선물 실패 위험 최소화 |

## 4. 핵심 기능 정의

### 4.1 기능 목록

| 번호 | 기능명 | 대상 | 우선순위 |
|------|--------|------|----------|
| F-01 | 회원가입 / 로그인 | 전체 | P0 |
| F-02 | 선물 이벤트 생성 | Giver | P0 |
| F-03 | 선물 옵션 등록 | Giver | P0 |
| F-04 | 수혜자 초대 링크 발송 | Giver | P0 |
| F-05 | 선물 선택 현황 확인 | Giver | P0 |
| F-06 | 구매 완료 처리 | Giver | P1 |
| F-07 | 초대장 확인 및 수락 | Recipient | P0 |
| F-08 | 선물 옵션 열람 및 선택 | Recipient | P0 |
| F-09 | 선택 완료 확인 | Recipient | P0 |
| F-10 | 내 정보 관리 | 전체 | P2 |

> **우선순위 기준**: P0 = 필수(MVP), P1 = 중요, P2 = 개선 사항

### 4.2 서비스 핵심 플로우

```
[Giver 플로우]
로그인 → 이벤트 생성 → 선물 옵션 등록 → 초대 링크 발송
       → 수혜자 선택 완료 알림 수신 → 선택 결과 확인 → 실제 구매 진행

[Recipient 플로우]
초대 링크 클릭 → 이벤트 정보 확인 → 초대 수락
              → 선물 옵션 목록 확인 → 원하는 선물 선택 → 선택 완료
```
